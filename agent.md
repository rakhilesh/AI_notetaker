# Agent Rules for AI Note Taker Project

1. **Git Workflow**: After completing every new feature or significant component, perform a `git add .` and `git commit -m "[feature name]: [description]"`.
2. **Aesthetics**: Maintain a premium "Glass IDE" look with glassmorphism, dark mode, and high-quality typography.
3. **OpenAI Integration**: Use OpenAI API for AI features. Provide a settings page for the API key.
4. **Chrome Context**: Shortcuts should be scoped to Chrome, not global.
5. **Documentation**: Include a detailed help page.
6. **Framework**: Use Vite + React for a responsive, modern component-based architecture.
